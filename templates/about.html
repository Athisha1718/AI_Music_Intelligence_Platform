ℹ️ About AI Music Intelligence Platform


The AI Music Intelligence Platform is a Deep Learning application that automatically analyzes music files and predicts their genre. It also generates visual audio representations and recommends similar songs based on extracted audio features.


🚀 Features

🎵

Genre Classification

Predicts the music genre using a trained CNN model.

🌊

Waveform Analysis

Displays the audio waveform of the uploaded song.

🌈

Spectrogram

Visualizes frequency information over time.

🎼

Chromagram

Shows pitch class energy distribution.

🧠

MFCC Heatmap

Displays MFCC features used by the CNN.

🎧

Song Recommendation

Finds similar songs using cosine similarity.

🧬

Music DNA

Shows Tempo, Energy, Brightness and other audio properties.

🤖

Deep Learning

Built using TensorFlow, Flask and the GTZAN Dataset.


🛠 Technologies Used

  • Python
  • Flask
  • TensorFlow / Keras
  • Librosa
  • NumPy
  • Pandas
  • Matplotlib
  • Scikit-learn
